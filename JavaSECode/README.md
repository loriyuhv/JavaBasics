### 1.2.1 什么是JDK、JRE

- **JDK**  (`J`ava `D`evelopment `K`it)：是Java程序开发工具包，包含`JRE` 和开发人员使用的工具。
- **JRE ** (`J`ava `R`untime `E`nvironment) ：是Java程序的运行时环境，包含`JVM` 和运行时所需要的`核心类库`。
- 小结：JDK = JRE + 开发工具集（例如Javac编译工具等）JRE = JVM + Java SE标准类库
