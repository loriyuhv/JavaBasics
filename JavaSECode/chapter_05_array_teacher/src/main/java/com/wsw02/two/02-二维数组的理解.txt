1. 二维数组的理解
- 对于二维数组的理解，可以看成是一维数组array1又作为另一个一维数组array的元素而存在。
- 其实，从数组的底层运行机制来看，其实没有多维数组。

3. 二维数组元素的默认初始值
3.1 动态初始化方式1：（比如：int[][] intArrays = new int[2][2];）
外层元素：默认存储地址值。
内层元素：默认与一维数组元素的不同类型的默认值规定相同。
3.2 动态初始化方式2：（比如：int[][] arrays = new int[2][];）
外层元素：null。
内层元素：不存在。如果调用会报错（NullPointerException）
